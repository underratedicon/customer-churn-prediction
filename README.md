# Customer Churn Prediction

This project compares Logistic Regression and Random Forest models for predicting customer churn using the IBM Telco Customer Churn dataset.

## Objective
The goal of this project is to identify customers who are likely to leave a telecom service and compare the performance of two machine learning models.

## Dataset
IBM Telco Customer Churn Dataset  
Source: Kaggle

## Models Used
- Logistic Regression
- Random Forest

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC Curve / AUC
- Confusion Matrix

## Results Summary
Both models achieved approximately 78% accuracy.  
Random Forest performed better in Recall, F1-score, and ROC-AUC (0.85), making it more effective for detecting churned customers.
